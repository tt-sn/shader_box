## シンプルな屈折shaderです．

## RefTexture.shader
貼り付けたテクスチャを屈折します．
_MainTex に貼り付けたいテクスチャを指定してください.

## RefBackgroundTexture.shader
オブジェクトを通過した背景を屈折します．
オブジェクトに貼り付けるだけで使えます.
